## Features

- **Authentication** – Secure login & registration with validation and toast feedback
- **Responsive Design** – Beautiful UI on mobile, tablet, and desktop
- **Dark Mode** – Toggle between light and dark themes
- **State Management** – Powered by **Zustand** for fast, scalable auth
- **Form Validation** – Real-time, field-level error handling
- **Toast Notifications** – Elegant success/error messages with **Sonner**
- **Next.js App Router** – File-system routing, server components, and streaming
- **TypeScript** – Full type safety and developer experience
- **Tailwind CSS** – Utility-first styling with design consistency
- **Lucide Icons** – Clean, modern SVG icons throughout

## Getting Started

### Prerequisites

- Node.js 18+
- npm / yarn / pnpm / bun

### Installation

```bash
git clone https://github.com/tiwariakrosh/blog-app.git
cd blog-app
npm install
# or
yarn install
# or
pnpm install
```
